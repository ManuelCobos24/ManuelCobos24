🎓 **Doble Graduado:** Cuento con las titulaciones de **Desarrollo de Aplicaciones Multiplataforma (DAM)** y **Desarrollo de Aplicaciones Web (DAW)**, lo que me permite tener una visión integral del ciclo de vida del software y las tecnologías web modernas.
